# CS:GO <=> QQ 消息通道

**开发中**

**message channel**是基于docker的，CS:GO服务器与QQ群双向信息通信框架。

## 特性
该项目提供的功能：
- [x] CSGO服务器与QQ群之间消息的稳定转发
- [x] 自己提供QQ号，消息转发安全可靠
- [x] 轻量化sourcemod插件
- [x] 可拓展的QQ插件，可以根据自己的需求更改
- [x] 使用Redis持久化保存服务器注册数据

更多的功能与使用方法见：[**csgowiki帮助中心**](https://docs.csgowiki.top/message-channel)
